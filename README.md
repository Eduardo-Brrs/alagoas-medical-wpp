# Alagoas Medical - Atendimento WhatsApp

Projeto para automatizar o atendimento inicial da Alagoas Medical via WhatsApp, começando com um menu rápido de triagem e evoluindo para uma aplicação mais completa com Java e Spring Boot.

## Objetivo
Reduzir o tempo gasto pela equipe no atendimento inicial, guiando o cliente por opções automáticas antes do contato final com o atendente.

## Estrutura
- `prototypes/menu-rapido`: versão simples e rápida para validação
- `backend`: sistema principal em Spring Boot
- `docs`: documentação do projeto

## MVP inicial
- Responder com menu automático
- Direcionar o cliente por opções
- Facilitar o trabalho da equipe

## Stack planejada
- Java
- Spring Boot
- Maven
- Git/GitHub
- WhatsApp API
