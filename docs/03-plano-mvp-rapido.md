# Plano MVP Rápido

## Objetivo
Entregar uma versão básica o mais rápido possível para ajudar a equipe no atendimento inicial.

## Fluxo inicial
1. Cliente envia mensagem
2. Sistema responde com saudação e menu
3. Cliente escolhe uma opção
4. Equipe recebe o cliente já mais direcionado

## Exemplo de menu
1 - Curativos e coberturas
2 - Terapia por pressão negativa
3 - Produtos hospitalares
4 - Falar com atendente
